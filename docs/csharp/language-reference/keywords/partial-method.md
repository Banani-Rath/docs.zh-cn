# 分部（方法）（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

分部方法在分部类型的一个部分中定义它的签名，并在该类型的另外一个部分中定义它的实现。  类设计人员可以使用分部方法提供由开发人员决定是否实现的方法挂钩（类似于事件处理程序）。  如果开发人员没有提供实现，则编译器会在编译时移除签名。  下列条件适用于分部方法：  
  
-   分部类型的两个部分中的签名必须匹配。  
  
-   方法必须返回 void。  
  
-   没有允许的访问修饰符。  分部方法是隐式私有的。
