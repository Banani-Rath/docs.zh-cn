# return（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`return` 语句终止它出现在其中的方法的执行并将控制返回给调用方法。  它还可以返回一个可选值。  如果方法为 `void` 类型，则可以省略 `return` 语句。  
  
 如果 return 语句位于 `try` 块中，则将在控制流返回到调用方法之前执行 `finally` 块（如果存在）。
