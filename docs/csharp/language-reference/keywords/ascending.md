# ascending（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`ascending` 上下文关键字用在查询表达式的 [orderby 子句](../../../csharp/language-reference/keywords/orderby-clause.md)中，用于指定从最小到最大的排序顺序。  因为 `ascending` 是默认排序顺序，所以您无须指定它。
