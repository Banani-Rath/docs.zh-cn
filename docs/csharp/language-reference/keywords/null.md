# null（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`null` 关键字是表示不引用任何对象的 null 引用的文字值。  `null` 是引用类型变量的默认值。  普通值类型不能为 null。  但是，C\# 2.0 引入了可以为 null 值的类型。  请参见 [可以为 null 的类型](../../../csharp/programming-guide/nullable-types/index.md)。
