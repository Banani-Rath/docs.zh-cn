# new 约束（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`new` 约束指定泛型类声明中的任何类型参数都必须有公共的无参数构造函数。  如果要使用 new 约束，则该类型不能为抽象类型。
