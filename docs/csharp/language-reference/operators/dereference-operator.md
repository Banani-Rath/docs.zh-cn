# -&gt; 运算符（C# 参考）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`->` 运算符将指针取消引用与成员访问组合在一起。  
  
## 备注  
 以下形式的表达式  
  
```  
x->y  
```  
  
 （其中 `x` 为 `T*` 类型的指针，`y` 为 `T` 的成员）等效于  
  
```  
(*x).y  
```  
  
 只能在标记为[不安全](../../../csharp/language-reference/keywords/unsafe.md)的代码中使用 `->` 运算符。  
  
 不能重载 `->` 运算符。
