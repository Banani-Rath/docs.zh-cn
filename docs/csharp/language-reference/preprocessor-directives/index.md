 虽然编译器没有单独的预处理器，本节中介绍的处理指令，就象一个。  它们在条件编译用于帮助。  与 C 和 C\+\+ 指令不同，不能使用这些指令创建宏。  
  
 预处理器指令必须是行上的唯一指令。
