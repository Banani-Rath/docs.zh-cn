# /nologo (C# Compiler Options)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

**\/nologo** 选项在编译器启动时取消显示登录版权标志，并在编译期间取消显示信息性消息。  
  
## 语法  
  
```  
/nologo  
```  
  
## 备注  
 此选项在开发环境中不可用；它仅在从命令行编译时才可用。  
  
 此编译器选项在 Visual Studio 中不可用，且不能通过编程方式进行更改。
