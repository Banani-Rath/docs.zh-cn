# 如何：串联多个字符串（C# 编程指南）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

串联是将一个字符串追加到另一个字符串末尾的过程。  使用 `+` 运算符串联字符串文本或字符串常量时，编译器会创建一个字符串。  串联不在运行时发生。  但字符串变量只能在运行时串联，  对此，您应该了解各种方法的性能含义。
