# 编译器错误 CS0709
“derived class”：无法从静态类“base class”进行派生  
  
 无法对静态类进行实例化或派生。 也就是说，静态类不能是任何其他类的基类。  
  
## 示例  
 下面的示例生成 CS0709。  
  
```  
// CS0709.cs // compile with: /target:library public static class Base {} public class Derived : Base {}   // CS0709  
```
