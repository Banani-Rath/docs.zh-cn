# 编译器错误 CS0589
内部编译器错误: 步骤“PARSE”  
  
 尝试确定编译器是否由于无法分析意外的语法而产生故障。 如果不是，请尝试 [与我们交流](/visual-studio/ide/talk-to-us)。
