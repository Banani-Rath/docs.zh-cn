# 编译器错误 CS1597
方法或访问器块后面的分号无效  
  
 不需要（或不允许）使用分号结束方法或访问器块。  
  
 下面的示例生成 CS1597：  
  
```  
// CS1597.cs class TestClass { public static void Main() { };   // CS1597, remove semicolon }  
```
