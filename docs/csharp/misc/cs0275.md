# 编译器错误 CS0275
“accessor”：可访问性修饰符不能在接口中的访问器上使用  
  
 对接口中的属性或索引器的某个访问器使用访问修饰符时，将出现此错误。 要解决此问题，请删除该访问修饰符。  
  
## 示例  
 下面的示例生成 CS0275：  
  
```  
// CS0275.cs public interface MyInterface { int Property { get; internal set;   // CS0275 } }  
```
