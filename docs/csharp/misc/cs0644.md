# 编译器错误 CS0644
“class1”无法从特殊类“class2”派生  
  
 类不能从任何以下基类中显式继承：  
  
-   **System.Enum**  
  
-   **System.ValueType**  
  
-   **System.Delegate**  
  
-   **System.Array**  
  
 这些均用作编译器的隐式基类。 例如，**System.ValueType** 是结构的隐式基类。  
  
 以下示例生成 CS0644：  
  
```  
// CS0644.cs class MyClass : System.ValueType   // CS0644 { }  
```
