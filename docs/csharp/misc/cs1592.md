# 编译器警告（等级 1）CS1592
所包含的注释文件中有格式错误的 XML \-\-“reason”  
  
 在由 [\<include\>](../../csharp/programming-guide/xmldoc/include.md) 标记指定的文件中发现了报告为 ***reason*** 的问题。
