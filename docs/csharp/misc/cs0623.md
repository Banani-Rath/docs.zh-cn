# 编译器错误 CS0623
数组初始值设定项只能在变量或字段初始值设定项中使用。 请尝试改用 new 表达式。  
  
 尝试在某个上下文中使用数组初始值设定项来初始化数组，而该上下文中不允许这样做。  
  
## 示例  
 下面的示例会生成 CS0623，因为编译器会将 \\{4\\} 解释为外部数组初始值设定项中的嵌入式数组初始值设定项：  
  
```  
//cs0632.cs using System; class X { public int[] x = { 2, 3, {4}}; //CS0623 }  
```
