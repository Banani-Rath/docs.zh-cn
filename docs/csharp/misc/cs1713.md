# 编译器错误 CS1713
为类型 Typename1 生成元数据名称时发生错误 \-\-“Reason”  
  
 此错误通常是由内部编译器错误引起的。 对代码进行一些小更改（如缩短名称长度），然后进行重新编译，可能会解决此问题。
