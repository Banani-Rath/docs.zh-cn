# 编译器错误 CS0710
静态类不能有实例构造函数  
  
 不能实例化静态类，因此它不需要构造函数。 若要避免此错误，请从静态类删除任何构造函数；如果你确实希望构造实例，请将此类转为非静态类。  
  
 下面的示例生成 CS0710：  
  
```  
// CS0710.cs public static class C { public C()  // CS0710 { } public static void Main() { } }  
```
