# 编译器警告（等级 1）CS1692
无效数字  
  
 多项预处理器指令（如 `#pragma` 和 `#line`）使用数字作为参数。 其中一个数字无效，原因是过大、格式错误、包含非法字符等。 若要更正此错误，请更正该数字。  
  
## 示例  
 以下示例生成 CS1692。  
  
```  
// CS1692.cs #pragma warning disable a  // CS1692 // Try this instad: // #pragma warning disable 1691 class A { static void Main() { } }  
```
