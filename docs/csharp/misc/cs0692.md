# 编译器错误 CS0692
重复的类型参数“identifier”  
  
 同一名称不能在类型参数列表中使用一次以上。 请重命名或删除重复的类型参数。  
  
## 示例  
 下面的示例生成 CS0692：  
  
```  
// CS0692.cs // compile with: /target:library class C <T, A, T>   // CS0692 { } class D <T, T>   // CS0692 { }  
```
