# 编译器错误 CS1013
无效数字  
  
 编译器检测到一个格式不正确的数字。 有关整型的详细信息，请参阅[整型表](../../csharp/language-reference/keywords/integral-types-table.md)。  
  
## 示例  
 下面的示例生成 CS1013：  
  
```  
// CS1013.cs class Sample { static void Main() { int i = 0x;   // CS1013 } }  
```
