# 编译器错误 CS0841
变量"name"在声明之前无法使用。  
  
 必须先声明名称，然后才能使用该变量。  
  
### 更正此错误  
  
1.  将变量声明移动到错误发生的行前。  
  
## 示例  
 以下示例生成 CS0841：  
  
```  
// cs0841.cs using System; public class C { public static int Main() { j = 5; // CS0841 int j; // To fix, move this line up. return 1; } }  
```
