# 编译器错误 CS0729
类型“type”是在此程序集中定义的，但又为它指定了一个类型转发器  
  
 类型转发器不能用于在同一程序集中定义的类型。  
  
## 示例  
 下面的示例生成 CS0729。  
  
```  
// CS0729.cs // compile with: /target:library using System.Runtime.CompilerServices; [assembly:TypeForwardedTo(typeof(TestClass))]   // CS0729 class TestClass {}  
```
