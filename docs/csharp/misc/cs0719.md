# 编译器错误 CS0719
“type”：数组元素不能是静态类型  
  
 静态类型的数组没有意义，因为数组元素是实例，但不能创建静态类型的实例。  
  
 下面的示例生成 CS0719：  
  
```  
// CS0719.cs public static class SC { public static void F() { } } public class CMain { public static void Main() { SC[] sca = new SC[10];  // CS0719 } }  
```
