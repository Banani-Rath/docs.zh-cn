# 访问应用程序窗体 (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`My.Forms` 对象为应用程序项目中声明的每个 Windows 窗体的实例提供了一种简单的访问方法。  也可以使用 `My.Application` 对象的属性访问应用程序的初始屏幕和主窗体，并获取应用程序的已打开窗体的列表。
