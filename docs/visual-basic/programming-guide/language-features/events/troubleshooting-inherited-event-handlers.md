## 过程  
  
#### 对于每次调用，事件处理程序中的代码执行两次  
  
-   继承的事件处理程序不得包含 [Handles](../../../../visual-basic/language-reference/statements/handles-clause.md) 子句。  基类中的方法已经与事件关联并将相应地激发事件。  从继承的方法中移除 `Handles` 子句。
