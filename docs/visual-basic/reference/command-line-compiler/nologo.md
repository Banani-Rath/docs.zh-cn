# /nologo (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

编译过程中禁止显示版权标志和通知消息。  
  
## 语法  
  
```  
/nologo  
```  
  
## 备注  
 如果指定 `/nologo`，则编译器将不显示版权标志。  默认情况下，`/nologo` 不生效。  
  
> [!NOTE]
>  `/nologo` 选项不能在 Visual Studio 开发环境内部使用，它仅在从命令行进行编译时可用。  
  
## 示例  
 下面的代码编译 `T2.vb` 并且不显示版权标志。  
  
```  
vbc /nologo t2.vb  
```
