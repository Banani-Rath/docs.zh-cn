# 项目已具有对程序集 &lt;assemblyidentity&gt; 的引用
项目已具有对程序集 \<assemblyidentity\> 的引用。 不能添加对“\<filepath\>”的第二个引用。  
  
 项目对同一程序集进行多个引用。  
  
 程序集标识包括程序集的名称、版本、公钥（如果有）和区域性。  
  
 此错误的一个可能原因是通过与原始引用不同的文件路径引用程序集的另一个副本。  
  
 **错误 ID：**BC32208  
  
### 更正此错误  
  
-   删除第二个引用。 它是不必要的，因为它引用同一程序集。
