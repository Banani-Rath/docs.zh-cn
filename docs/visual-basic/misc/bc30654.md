# Function、Get 或 Operator 中的“Return”语句必须返回值
`Return` 语句必须用于返回调用过程的值。 不能使用 `Return` 语句本身来控制程序流。  
  
 **错误 ID：**BC30654  
  
### 更正此错误  
  
1.  指定可返回函数或过程的值。  
  
2.  使用 `End` 语句，以使程序退出当前过程。
