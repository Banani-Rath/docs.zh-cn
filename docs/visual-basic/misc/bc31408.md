# “&lt;specifier1&gt;”不能与“&lt;specifier2&gt;”组合
同时使用了两个不兼容的说明符（如 `Shadows` 和 `Overrides`）。  
  
 **错误 ID：**BC31408  
  
### 更正此错误  
  
-   删除其中一个说明符。
