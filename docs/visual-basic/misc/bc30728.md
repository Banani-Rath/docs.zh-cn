# 结构中声明的方法不能包含“Handles”子句
结构方法不能使用 `Handles` 关键字来处理事件。  
  
 **错误 ID：**BC30728  
  
### 更正此错误  
  
-   请考虑重新设计作为类的结构。  
  
     可以使用 `AddHandler` 将事件与事件处理程序关联在结构中，前提是该结构实现的是接口中定义的事件处理程序。
