# 字段“&lt;fieldname&gt;”的类型不受支持
字段声明为 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 不能表示的类型。  
  
 **错误 ID：**BC30656  
  
### 更正此错误  
  
1.  将字段定义为支持的类型。  
  
2.  开发一种可以描述字段中数据的新类型。
