# 用 &quot;As&quot; 子句声明的事件必须有委托类型
指定了非委托的类型。  
  
 **错误 ID：**BC31044  
  
### 更正此错误  
  
1.  删除 `As` 子句。  
  
2.  使用 `As` 子句指定委托。
