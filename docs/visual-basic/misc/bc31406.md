# “&lt;methodname&gt;”不能重写已被隐藏的方法
某个方法试图重写隐藏其他某个成员的方法。  
  
 **错误 ID：**BC31406  
  
### 更正此错误  
  
-   使用 `Shadows` 修饰符重写隐藏成员。
