# 未能找到库“&lt;libraryname&gt;”
未能找到指定的库。  
  
 当从命令提示符下进行编译，并使用引用多个程序集的响应 \(.rsp\) 文件时，可能会发生此问题  
  
 **错误 ID：**BC2017  
  
### 更正此错误  
  
-   检查库名称以确保其正确无误。
