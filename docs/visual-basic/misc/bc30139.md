# 设置程序集清单选项时出错：&lt;error message&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 编译器调用程序集链接器（Al.exe，也称作 Alink）生成包含清单的程序集。 该链接器报告了查找密钥文件或查找用来对程序集签名的密钥名称时发生的错误。  
  
 **错误 ID：**BC30139  
  
### 更正此错误  
  
1.  检查引用的错误信息并参考名为[“Al.exe 工具错误和警告”](http://msdn.microsoft.com/zh-cn/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)的页面，以获得进一步的解释和建议。  
  
2.  如果仍然出现错误，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
