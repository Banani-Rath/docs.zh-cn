# “&lt;typename&gt;”是不受支持的类型
指定的类型不能通过 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 表示。  
  
 **错误 ID：**BC30649  
  
### 更正此错误  
  
-   使用受支持的类型。  
  
-   开发新类型。
