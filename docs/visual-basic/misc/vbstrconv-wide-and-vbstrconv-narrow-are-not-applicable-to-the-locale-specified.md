# VbStrConv.Wide 和 VbStrConv.Narrow 不适用于指定的区域设置
应用程序尝试使用 `VbStrConv` 枚举成员 `Wide` 或 `Narrow`，它们不适用于指定的区域设置。  
  
### 更正此错误  
  
1.  删除 `VbStrConv.Wide` 或 `VbStrConv.Narrow`。
