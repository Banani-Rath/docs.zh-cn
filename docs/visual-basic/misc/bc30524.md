# 属性“&lt;propertyname&gt;”为“WriteOnly”
已尝试从声明 `WriteOnly` 的属性中检索值。  
  
 **错误 ID：**BC30524  
  
### 更正此错误  
  
-   从属性声明中删除 `WriteOnly` 说明符。
