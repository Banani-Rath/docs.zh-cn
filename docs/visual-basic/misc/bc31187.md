# 命名空间声明必须以“xmlns”开头
已指定的 XML 命名空间没有需要的 `xmlns` 标识符。`xmlns` 标识符必须仅包含小写字符。  
  
 **错误 ID：**BC31187  
  
### 更正此错误  
  
-   声明 XML 命名空间时使用 `xmlns` 标识符。 下面是一个示例：  
  
    ```vb#  
    Imports <xmlns:ns="http://SampleNamespace">  
    ```
