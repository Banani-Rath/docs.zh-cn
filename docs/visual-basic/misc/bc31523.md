# “System.Runtime.InteropServices.DllImportAttribute”不能应用于“Declare”。
`Declare` 函数已使用了 `DllImportAttribute` 特性。 此特性仅可用于 `Function` 或 `Sub`。  
  
 **错误 ID:** BC31523  
  
### 更正此错误  
  
1.  请从此 `Declare` 语句中删除 `DllImportAttribute` 特性。
