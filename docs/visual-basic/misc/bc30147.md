# “Return”语句在“即时”窗口中无效
`Return` 语句执行分支，但不允许用于调试上下文。  
  
 **错误 ID：**BC30147  
  
### 更正此错误  
  
-   不要在“即时”窗口中发出 `Return` 语句。
