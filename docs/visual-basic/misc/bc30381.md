# “Finally”只能在“Try”语句中出现一次
`Finally` 用于完成 `Try...Catch...Finally` 块；因此它只能在块的末尾出现一次。  
  
 **错误 ID：**BC30381  
  
### 更正此错误  
  
1.  找到并删除冗余 `Finally`。
