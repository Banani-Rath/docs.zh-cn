# Lambda 表达式无法转换为“&lt;类型名称&gt;”，因为类型“&lt;类型名称&gt;”被声明为“MustInherit”，无法创建
一个语句尝试将 lambda 表达式转换为一个仅可为基类且不可用于创建实例的类型。  
  
 **错误 ID：**BC36626
