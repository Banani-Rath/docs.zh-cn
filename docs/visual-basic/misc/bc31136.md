# “AddHandler”和“RemoveHandler”方法参数必须与包含事件具有相同的委托类型。
`Custom Event` 声明必须具有 `AddHandler` 或 `RemoveHandler` 声明，其中每个采用由自定义事件的 `As` 子句所指定的委托类型的单个参数。  
  
 **错误 ID：**BC31136  
  
### 更正此错误  
  
-   将参数的类型改为与自定义事件的 `As` 子句所指定的委托类型相同的类型。
