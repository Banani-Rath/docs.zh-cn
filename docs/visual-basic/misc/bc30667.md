# ParamArray 参数必须声明为“ByVal”
`ParamArray` 参数不能使用 `ByRef` 修饰符。  
  
 **错误 ID：**BC30667  
  
### 更正此错误  
  
-   使用 `ByVal` 修饰符声明 `ParamArray` 参数。
