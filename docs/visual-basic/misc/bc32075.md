# 泛型类型或泛型类型中包含的类型中不允许使用“Declare”语句
`Declare` 语句是泛型类或结构的一部分，或某个类或结构在泛型类或结构中声明。  
  
 Visual Basic 和 .NET Framework 当前不支持外部引用和泛型类型的任意组合。 编译器需要外部过程的所有参数和返回类型才能正确调用。  
  
 **错误 ID:** BC32075  
  
### 更正此错误  
  
-   将 `Declare` 语句移到所有泛型类型的作用域之外，或者全部删除。
