# “&lt;membername&gt;”已由为此 &lt;containername&gt; 生成的“&lt;procedurename&gt;”声明
所声明的某个成员与由公共语言运行时声明的某个成员同名。  
  
 **错误 ID：**BC30733  
  
### 更正此错误  
  
-   更改成员名称以消除冲突。
