# &lt;Property&gt; 参数不能声明为 &quot;ByRef&quot;
属性参数必须使用 `ByVal` 修饰符进行声明。  
  
 **错误 ID：**BC30651  
  
### 更正此错误  
  
-   将属性参数定义为 `ByVal`。
