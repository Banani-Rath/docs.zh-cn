# 无法打开资源文件“&lt;filename&gt;”：&lt;error&gt;
打开指定的资源文件时出错。  
  
 **错误 ID：**BC31509  
  
### 更正此错误  
  
1.  验证指定的名称和位置是否正确。  
  
2.  验证该文件是否已损坏。
