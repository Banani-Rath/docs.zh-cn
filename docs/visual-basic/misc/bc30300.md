# “&lt;method1&gt;”和“&lt;method2&gt;”不能相互重载，因为它们的差异仅在于可选参数
已尝试使用与前者区别仅在于可选参数的另一种方法重载某种方法。 具有可选参数的方法相当于两个重载方法，其中一个有可选参数，而另一个没有。 因此，无法使用对应其中任一方法的参数列表来重载方法。  
  
 **错误 ID：**BC30300  
  
### 更正此错误  
  
-   确保这两种方法的区别不仅在于可选参数。
