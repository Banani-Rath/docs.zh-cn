# “&lt;procedurename&gt;”有不受支持的返回类型或不受支持的参数类型
过程使用不能由 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 表示的参数或返回类型声明。  
  
 **错误 ID：**BC30657  
  
### 更正此错误  
  
-   将返回类型或参数定义为受支持的类型。  
  
-   开发用于描述此过程所使用数据的新类型。
