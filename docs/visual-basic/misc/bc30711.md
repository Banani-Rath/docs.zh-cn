# 无法设置不是位于堆栈顶部的方法的局部变量值
如果变量位于调用堆栈的顶部，你才能对其修改。 例如，如果 `procedure1` 调用 `procedure2` 且你处于 `procedure1`，则不能修改 `procedure2` 中的变量。  
  
 **错误 ID：**BC30711  
  
### 更正此错误  
  
-   修改位于调用堆栈顶部的变量。
