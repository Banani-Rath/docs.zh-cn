# “&lt;elementname&gt;”不是“&lt;typename&gt;”中定义的扩展方法“&lt;methodname&gt;”的参数
过程调用提供一个不是由该扩展方法定义的附加参数。  
  
 **错误 ID:** BC36585  
  
### 更正此错误  
  
-   从参数列表中删除多余的参数。
