# 不能组合 SimplifiedChinese 和 VbStrConv.TraditionalChinese
应用程序尝试组合 `VbStrConv` 枚举成员 `SimplifiedChinese` 和 `TraditionalChinese`，而它们是互斥的。  
  
### 更正此错误  
  
-   删除  `VbStrConv.SimplifiedChinese` 或 `VbStrConv.TraditionalChinese`。
