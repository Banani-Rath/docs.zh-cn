# 表达式或语句在调试窗口中无效。
无法在调试上下文中使用 `Try...Catch...Finally` 语句。  
  
 **错误 ID：**BC30706  
  
### 更正此错误  
  
1.  从调试代码中删除 `Try...Catch...Finally` 语句。
