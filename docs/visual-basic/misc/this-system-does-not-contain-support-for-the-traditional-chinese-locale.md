# 此系统不包含对繁体中文区域设置的支持
你尝试使用 `VbStrConv.TraditionalChinese` 枚举成员，但你的系统不支持繁体中文区域设置。  
  
### 更正此错误  
  
1.  升级计算机以支持繁体中文区域设置，或在支持该区域设置的计算机上运行该应用程序。
