# 在“&lt;namespace&gt;”：&lt;message&gt; 中多次声明了“Sub Main”
仅可在命名空间内声明一次 `Sub Main`。  
  
 **错误 ID：**BC30738  
  
### 更正此错误  
  
-   请确保项目中只有一个 `Sub Main` 过程。
