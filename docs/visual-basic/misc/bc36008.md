# “Using”必须以匹配的“End Using”结束
出现 `Using` 语句而没有相应的 `End Using` 语句。  
  
 必须使用 `End Using` 语句结束 `Using` 块。  
  
 **错误 ID：**BC36008  
  
### 更正此错误  
  
1.  如果此 `Using` 块属于一组嵌套的 `Using` 块，请确保每个块均已正确终止。  
  
2.  将 `End Using` 语句添加到 `Using` 块末尾。
