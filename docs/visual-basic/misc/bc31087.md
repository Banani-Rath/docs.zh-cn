# 不能在变量及其类型上同时指定数组修饰符
在变量及其类型中同时存在数组修饰符，这表示数组的数组。  
  
 **错误 ID：**BC31087  
  
### 更正此错误  
  
-   从类型说明符中删除数组修饰符。
