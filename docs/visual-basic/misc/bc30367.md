# 无法为类“&lt;classname&gt;”编制索引，因为它没有默认属性。
你无法将默认属性提供给类。  
  
 **错误 ID：**BC30367  
  
### 更正此错误  
  
-   通过使用 `Default` 关键字作为其声明的开头来指定一个属性作为默认属性。
