# 通过“编辑并继续”创建的对象尚不能用于求值
在 Visual Studio 调试器中，正在创建新对象，但尚无法访问它。  
  
 对象不可用的可能原因包括内存分配失败和对象状态不同步。 这些是临时内部状况，通常不是调试器输入导致的。  
  
 **错误 ID：**BC30958  
  
### 更正此错误  
  
-   终止并重新启动调试。
