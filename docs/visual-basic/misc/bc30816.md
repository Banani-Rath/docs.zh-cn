# 未声明“&lt;成员名称&gt;”
未声明“`<membername>`”。`Debug` 对象功能在 `System` 程序集中的 `System.Diagnostics.Debug` 中可用。  
  
 无法找到指定的成员。  
  
 **错误 ID：**BC30816  
  
### 更正此错误  
  
1.  检查成员的拼写。  
  
2.  使用 `Imports` 语句或完全限定在其他命名空间中定义的成员。 例如，在 `System.Diagnostics.Debug` 命名空间中声明 `WriteLine` 函数。
