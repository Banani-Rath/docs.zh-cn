# “Custom”修饰符只能在紧靠“Event”声明之前使用
在自定义事件声明中，`Custom` 关键字必须仅在 `Event` 关键字之前。  
  
 **错误 ID：**BC31140  
  
### 更正此错误  
  
-   移动 `Custom` 关键字，使其位于 `Event` 关键字之前。  
  
     \- 或 \-  
  
-   删除 `Custom` 关键字。
