# “GoTo”语句在即时窗口中无效
`GoTo` 语句执行分支，但不允许用于调试上下文。  
  
 **错误 ID：**BC30133  
  
### 更正此错误  
  
-   不要在“即时”窗口中发出 `GoTo` 语句。
