# “RaiseEvent”语句在“即时”窗口中无效
无法在“即时”窗口中使用 `RaiseEvent` 语句。  
  
 **错误 ID：**BC30739  
  
### 更正此错误  
  
-   从“代码编辑器”中的代码测试 `RaiseEvent` 语句。
