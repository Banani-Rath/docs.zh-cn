# 无法处理事件“&lt;eventname&gt;”，因为不能从“&lt;name&gt;”访问它
你试图处理不可访问的事件。 例如，如果 `Handles` 变量共享，则处理事件的方法也必须共享。  
  
 **错误 ID：**BC30585  
  
### 更正此错误  
  
1.  请确保该事件可访问。
