# “SyncLock”语句必须以匹配的“End SyncLock”结尾
`SyncLock` 块以 `SyncLock` 关键字开头，以 `End SyncLock` 构造结尾。  
  
 **错误 ID：**BC30675  
  
### 更正此错误  
  
-   请确保 `SyncLock` 块以 `End SyncLock` 构造结尾。
