# “Try”必须以匹配的“End Try”结束
`Try` 用于启动 `Try` 块；因此它只能出现在块的开头，并以匹配的`End Try` 语句结束块。 存在冗余 `Try`，或未以 `Finally` 结束 `Try` 块。  
  
 **错误 ID：**BC30384  
  
### 更正此错误  
  
1.  查找并删除无关的 `Try`，或以匹配的 `End Try` 结束块。
