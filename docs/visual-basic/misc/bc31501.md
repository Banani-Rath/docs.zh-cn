# “ReadOnly”特性属性“&lt;propertyfield&gt;”不能作为赋值的目标
尝试给特性中的 `ReadOnly` 属性赋值。  
  
 **错误 ID：**BC31501  
  
### 更正此错误  
  
1.  删除属性赋值语句。  
  
2.  如果使用你开发的属性，请从特性属性中删除 `ReadOnly` 或 `Shared` 修饰符。
