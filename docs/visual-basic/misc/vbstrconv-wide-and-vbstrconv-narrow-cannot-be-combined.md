# 不能组合 VbStrConv.Wide 和 VbStrConv.Narrow
你的应用程序尝试组合 `VbStrConv` 枚举成员 `Wide` 和 `Narrow`，而它们是互斥的。  
  
### 更正此错误  
  
1.  删除 `VbStrConv.Wide` 或 `VbStrConv.Narrow`。
