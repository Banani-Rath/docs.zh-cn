# 命名空间中类型的完全限定名不能超过 &lt;number&gt; 个字符。
该类型的完全限定名不能超过指定数目的字符。  
  
 **错误 ID：**BC30031  
  
### 更正此错误  
  
-   缩短类型的名称。
