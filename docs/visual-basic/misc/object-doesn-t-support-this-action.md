# 对象不支持此操作 (Visual Basic)
你引用了不受此对象支持的方法或属性。  
  
### 更正此错误  
  
1.  有关对象的详细信息，请阅读对象的文档。  
  
2.  确保属性和方法的名称拼写正确。
