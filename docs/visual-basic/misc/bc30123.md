# “End”语句在“即时”窗口中无效
`Stop` 和 `End` 语句暂停执行，且不允许在调试上下文中使用它们。  
  
 **错误 ID：**BC30123  
  
### 更正此错误  
  
-   不要在“即时”窗口中发出 `End` 或 `Stop` 语句。
