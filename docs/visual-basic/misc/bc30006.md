# 需要对模块“&lt;modulename&gt;”（包含事件“&lt;eventname&gt;”的定义）的引用
需要对模块“\<`modulename`\>”（包含事件“\<`eventname`\>”的定义）的引用。 请向项目中添加一个。  
  
 事件在项目不直接引用的模块中定义。 如果事件在多个模块中定义，则 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 编译器需要一个引用以避免多义性。  
  
 **错误 ID：**BC30006  
  
### 更正此错误  
  
-   将未引用模块的名称包括在项目引用中。
