# “Exit Do”只能出现在“Do”语句内
`Exit Do` 语句出现在 `Do` 循环外部。`Exit Do` 仅在 `Do` 语句和相应的 `Loop` 语句之间有效。  
  
 **错误 ID：**BC30089  
  
### 更正此错误  
  
1.  确保 `Exit Do` 之前有一个有效的 `Do` 语句，之后有一个有效的 `Loop` 语句。  
  
2.  验证 `Do` 循环内的其他控制结构是否被正确终止。
