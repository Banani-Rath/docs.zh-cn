# “&lt;name&gt;”不能通过其默认实例引用自身，请改用“Me”
尝试从窗体内部将该窗体作为默认实例引用。 这可能导致窗体递归调用自身。  
  
 在大多数情况下，当引用该窗体的当前实例时，应使用 `Me` 而不是默认实例。  
  
 **ID：**BC31139  
  
### 更正此错误  
  
-   使用 `Me` 来引用对象。
