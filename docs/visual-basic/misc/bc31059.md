# “&lt;eventname&gt;”隐式定义了“&lt;membername&gt;”，这与在&lt;type&gt;“&lt;typename&gt;”中隐式声明的某个成员冲突
类型成员名称与为事件隐式创建的成员名称冲突。 事件隐式创建了多个变量。 例如，声明 `Event X` 隐式声明了名称 `XEventHandler`、`XEvent`、`add_X` 和 `remove_X`。  
  
 **错误 ID：**BC31059  
  
### 更正此错误  
  
-   重命名显式声明的成员，以消除命名冲突。
