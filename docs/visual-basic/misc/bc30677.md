# “AddHandler”或“RemoveHandler”语句事件操作数必须是以点限定的表达式或者是简单的名称。
为 `AddHandler` 或 `RemoveHandler` 的事件参数指定的项未被识别为事件。  
  
 **错误 ID：**BC30677  
  
### 更正此错误  
  
-   指定引发事件的对象名称，后跟点 \(`.`\) 和事件名称，如以下示例所示。
