# 已声明 &quot;Set&quot;
代码块中多次出现 `Set` 语句。`Set` 语句声明用于为某个属性赋值的 `Set` 属性过程。  
  
 **错误 ID：**BC30444  
  
### 更正此错误  
  
-   删除多余的 `Set` 语句。
