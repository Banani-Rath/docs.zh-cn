# “Optional”不能与“ParamArray”组合
`Optional` 和 `ParamArray` 修饰符互相排斥。  
  
 **错误 ID：**BC30642  
  
### 更正此错误  
  
-   指定 `ParamArray` 或 `Optional`，但不能同时指定这两者。
