# 只能指定 &quot;NotOverridable&quot;、&quot;MustOverride&quot; 或 &quot;Overridable&quot; 中的一个
在元素声明中发生重写规范冲突。 重写关键字互斥。  
  
 **错误 ID：**BC30177  
  
### 更正此错误  
  
-   选择一个重写说明符，并将其他关键字从声明中删除。
