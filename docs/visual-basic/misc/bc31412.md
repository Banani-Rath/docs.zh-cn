# 类中的“Handles”必须指定用单个标识符限定的“WithEvents”变量、“MyBase”、“MyClass”或“Me”
若要指定事件处理程序，`Handles` 语句必须指定用 `WithEvents` 关键字声明的对象变量，或通过 `MyBase` 关键字限定的成员。  
  
 **错误 ID：**BC31412  
  
### 更正此错误  
  
1.  使用 `WithEvents` 修饰符来声明要用于 `Handles` 语句的变量。  
  
2.  为基类中的当前类指定事件名称。
