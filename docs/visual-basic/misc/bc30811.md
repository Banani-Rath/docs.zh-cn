# “ReDim”语句不能再用于声明数组变量
`ReDim` 仅可用于更改现有数组的大小。  
  
 **错误 ID：**BC30811  
  
### 更正此错误  
  
-   在声明数组时指定其大小；例如：  
  
    ```  
    Dim X(20) As Integer  
    ```
