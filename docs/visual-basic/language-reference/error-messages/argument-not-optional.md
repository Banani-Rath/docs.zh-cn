# 非可选参数 (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

参数的个数和类型必须符合要求。  参数个数不对，或省略的参数非可选参数。  只有在过程定义中声明为 `Optional` 的参数才能在调用时省略。  
  
### 更正此错误  
  
1.  提供所有必需的参数。  
  
2.  确保省略的参数是可选的。  否则，应在调用时提供该参数，或者在定义中将该参数声明为 `Optional`。
