# 字符串常量必须以双引号结束
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

字符串常数必须以双引号开始和结束。  
  
 **错误 ID：**BC30648  
  
### 更正此错误  
  
-   请确保字符串文本以引号 \("\) 结束。  如果从其他文本编辑器中粘贴值，请确保粘贴的字符是有效的双引号，而不是其他类似双引号的字符，如“智能”或“弯”引号（" 或 "）或两个单引号 \(''\)。
