# “AddressOf”操作数必须是方法名（不带括号）
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`AddressOf` 运算符创建引用特定过程的过程委托实例。  语法如下所示。  
  
 `AddressOf` `procedurename`  
  
 在 `AddressOf` 后面的参数两边插入了括号，但这里不需要括号。  
  
 **错误 ID：**BC30577  
  
### 更正此错误  
  
1.  移除 `AddressOf` 后面的参数两边的括号。  
  
2.  确保参数是方法名。
