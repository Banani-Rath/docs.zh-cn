# 无法加载类“&lt;类名&gt;”的信息
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

引用了不可用的类。  
  
 **错误 ID：**BC30712  
  
### 更正此错误  
  
1.  验证定义了该类并且类名称的拼写正确。  
  
2.  尝试访问该模块中声明的一个成员。  在某些情况下，因为尚未加载在其中声明成员的模块，所以调试环境不能定位成员。
