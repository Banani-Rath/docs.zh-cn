# 由于内部系统错误，无法获得串行端口名称
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

调用 `My.Computer.Ports.SerialPortNames` 属性时发生内部错误。  
  
### 更正此错误  
  
1.  有关故障排除的详细信息，请参阅[调试器基础知识](/visual-studio/debugger/debugger-basics)。  
  
2.  记录发生错误的环境，并与 Microsoft 产品支持服务联系。
