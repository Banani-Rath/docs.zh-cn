# 权限被拒绝 (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

尝试写入具有写保护的磁盘或访问锁定文件。  
  
### 更正此错误  
  
1.  若要打开具有写保护的文件，请更改文件的写保护特性。  
  
2.  确保没有其他进程锁定文件，并等待该文件释放后再打开该文件。  
  
3.  若要访问注册表，请检查您的用户权限是否包括这种类型的注册表访问。
