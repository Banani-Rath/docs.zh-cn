# “Module”语句只能出现在文件级或命名空间级
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`Module` 语句必须出现在源文件顶部并紧跟在 `Option` 和 `Imports` 语句、全局特性及命名空间声明之后，但必须在其他所有声明之前。  
  
 **错误 ID：**BC30617  
  
### 更正此错误  
  
-   将 `Module` 语句移动到命名空间声明或源文件的顶部。
