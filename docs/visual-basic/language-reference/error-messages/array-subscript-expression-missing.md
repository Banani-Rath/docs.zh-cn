# 缺少数组下标表达式
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

数组初始化遗漏了一个或多个定义数组界限的下标。  例如，语句可能包含遗漏了第三个下标的 `myArray (5,5,,10)` 表达式。  
  
 **错误 ID：**BC30306  
  
### 更正此错误  
  
-   提供丢失的下标。
