# 自动错误
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

执行方法或者获取或设置对象变量的属性时发生错误。  错误由创建该对象的应用程序报告。  
  
### 更正此错误  
  
1.  检查 `Err` 对象的属性，以确定错误来源和错误性质。  
  
2.  在紧邻访问语句之前使用 `On Error Resume Next` 语句，然后检查紧邻访问语句之后是否存在错误。
