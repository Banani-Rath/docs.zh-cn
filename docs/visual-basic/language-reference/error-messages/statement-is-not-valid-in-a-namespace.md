# 语句在命名空间中无效
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

该语句不能出现在命名空间级别。  可以在命名空间级出现的声明只有模块、接口、类、委托、枚举以及结构声明。  
  
 **错误 ID：**BC30001  
  
### 更正此错误  
  
-   将该语句移到模块、类、接口、结构、枚举或委托定义中的某个位置。
