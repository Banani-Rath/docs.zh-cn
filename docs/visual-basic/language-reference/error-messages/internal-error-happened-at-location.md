# 在 &lt;位置&gt; 处发生的内部错误
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

发生了内部错误。  该内部错误发生时所在的行包含在错误信息中。  
  
### 更正此错误  
  
-   确保此错误不是由 `Error` 语句或 `Raise` 方法生成的；如果不是，请与 Microsoft 产品支持服务部门联系以报告该消息出现的条件。
