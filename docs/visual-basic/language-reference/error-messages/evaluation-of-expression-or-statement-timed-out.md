# 表达式或语句的计算已超时
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

表达式的计算未能及时完成。  
  
 **错误 ID：**BC30722  
  
### 更正此错误  
  
1.  请验证输入的代码是否正确。  
  
2.  简化表达式，使其以较少的时间执行。
