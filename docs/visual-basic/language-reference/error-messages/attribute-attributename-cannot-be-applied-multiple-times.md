# 特性“&lt;特性名&gt;”不能应用多次
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

此特性只能应用一次。  `AttributeUsage` 特性决定了某个特性是否可以应用多次。  
  
 **错误 ID：**BC30663  
  
### 更正此错误  
  
1.  确保特性只应用一次。  
  
2.  如果使用的是自已开发的自定义特性，请考虑更改其 `AttributeUsage` 特性，以允许特性使用多次，如下面的示例所示。  
  
    ```  
    <AttributeUsage(AllowMultiple := True)>  
    ```
